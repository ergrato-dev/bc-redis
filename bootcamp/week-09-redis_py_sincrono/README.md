# Semana 09 — redis-py Síncrono — Patrones de Aplicación

Integra Redis en aplicaciones Python síncronas: ConnectionPool, caching con decoradores, session store y rate limiting.

---

## 🎯 Objetivos de Aprendizaje

Al finalizar esta semana, el estudiante será capaz de:

- ConnectionPool: configuración y gestión de conexiones
- Serialización: JSON vs Pickle — trade-offs
- Patrón Caching: cache-aside con decoradores Python
- Session Store: almacenamiento de sesiones con TTL
- Rate Limiting: sliding window y token bucket con Redis
