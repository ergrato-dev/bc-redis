# Semana 08 — Pipelining y Benchmarking

Optimiza el rendimiento con pipelining, reduce round-trips y mide el throughput con redis-benchmark.

---

## 🎯 Objetivos de Aprendizaje

Al finalizar esta semana, el estudiante será capaz de:

- Pipelining: batching de comandos sin garantía de atomicidad
- redis-py pipeline() en modo buffered y transaccional
- redis-benchmark: flags esenciales, interpretación de resultados
- OBJECT ENCODING: entender cómo Redis almacena internamente
- Slow log: SLOWLOG GET / SLOWLOG RESET / SLOWLOG LEN
