# Semana 05 — Pub/Sub

Implementa comunicación en tiempo real con el mecanismo Pub/Sub de Redis: patrones fan-out y mensajería fire-and-forget.

---

## 🎯 Objetivos de Aprendizaje

Al finalizar esta semana, el estudiante será capaz de:

- PUBLISH / SUBSCRIBE / UNSUBSCRIBE
- PSUBSCRIBE / PUNSUBSCRIBE (pattern subscriptions)
- Ciclo de vida de un subscriber en redis-py
- Casos de uso: notificaciones en tiempo real, chat, eventos de sistema
- Limitaciones del Pub/Sub: sin persistencia, sin historial
