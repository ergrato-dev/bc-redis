# Semana 03 — Sets

Aprende la estructura Set de Redis y sus poderosas operaciones de conjuntos para dedupliación y relaciones.

---

## 🎯 Objetivos de Aprendizaje

Al finalizar esta semana, el estudiante será capaz de:

- SADD / SMEMBERS / SCARD / SISMEMBER
- SINTER / SUNION / SDIFF y sus variantes *STORE
- SRANDMEMBER / SPOP para selecciones aleatorias
- Casos de uso: tags, relaciones sociales, deduplicación
- SSCAN para conjuntos grandes
