# Semana 10 — redis-py Asíncrono — Locks, Queues y Leaderboards

Implementa patrones avanzados con redis.asyncio: distributed locks, job queues y leaderboards en tiempo real.

---

## 🎯 Objetivos de Aprendizaje

Al finalizar esta semana, el estudiante será capaz de:

- redis.asyncio: AsyncRedis, async pipelines, context managers
- Distributed Lock: algoritmo Redlock, safety y liveness
- Job Queue: implementación con Lists y Streams
- Leaderboard en tiempo real con Sorted Sets
- Testing async con fakeredis.aioredis y pytest-asyncio
