# Semana 12 — Producción y Proyecto Final Integrador

Aplica seguridad avanzada, performance tuning y entrega el proyecto final que integra todo el bootcamp.

---

## 🎯 Objetivos de Aprendizaje

Al finalizar esta semana, el estudiante será capaz de:

- Seguridad avanzada: ACLs (ACL SETUSER), TLS, rename-command
- Performance tuning: maxmemory, eviction policies, slow log
- Proyecto final: sistema completo con múltiples estructuras de datos
- Code review y presentación de proyectos
- Hoja de ruta post-bootcamp: Redis Enterprise, Redis Cloud, certificaciones

---

- Haber completado la [Semana 11](../week-11-alta_disponibilidad/README.md)
- Tener el entorno Docker funcionando ([Setup](../../../docs/setup/README.md))
