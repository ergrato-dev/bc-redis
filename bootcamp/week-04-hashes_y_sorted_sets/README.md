# Semana 04 — Hashes, Sorted Sets, Persistencia y Seguridad

Modela entidades con Hashes, construye leaderboards con Sorted Sets y configura persistencia y seguridad básica.

---

## 🎯 Objetivos de Aprendizaje

Al finalizar esta semana, el estudiante será capaz de:

- HSET / HGET / HGETALL / HDEL / HINCRBY / HMSET
- ZADD / ZRANGE / ZRANK / ZSCORE / ZRANGEBYSCORE / ZRANGEBYLEX
- Persistencia: RDB (BGSAVE) y AOF (appendonly)
- Seguridad básica: requirepass, bind, rename-command
- redis.conf: configuración completa comentada
