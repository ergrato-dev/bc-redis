# Semana 06 — Redis Streams

Domina Redis Streams como mecanismo de mensajería persistente con consumer groups y entrega at-least-once.

---

## 🎯 Objetivos de Aprendizaje

Al finalizar esta semana, el estudiante será capaz de:

- XADD / XREAD / XRANGE / XREVRANGE / XLEN
- Consumer Groups: XGROUP CREATE / XREADGROUP / XACK
- XPENDING / XCLAIM para mensajes no confirmados
- XTRIM y políticas de retención
- Comparación Streams vs Pub/Sub: cuándo usar cada uno

---
